# wapp_file_manager

Файловый менеджер на flask. 

- Есть просмотр изображений
- Подсветка кода тестовых файлов
- Есть просмотр PDF
- Есть просмотр DJVU с конвертацией в PDF

![](screenshots/2022-12-17_18-16.png)

## Запуск

```
python3 release.zip
# или
python3 __main__.py
```

## Зависимости

- apt пакеты
    - djvulibre-bin
    - unoconv
